# Features
* Plays the stereo test voice in
  * App
  * Quick Settings Tile
  * Widget
* Plays sound of a specific frequency.
* Plays frequency sweep.
* Shows definitions of left/right.
* Supports
  * Dark theme (Android 10+)
  * Many languages
